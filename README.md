## pyadb
Tool for easy navigation using adb command line tool

### setup 
#### mac/linux
```
cd pyadb/
python -m venv env
source env/bin/activate
pip install -r r.txt
```
#### windows
```
cd pyadb/
python -m venv env
cd env/Scripts
activate
pip install -r r.txt
```
